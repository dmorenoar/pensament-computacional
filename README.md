## 📋 Descripció

Lloc web de presentació d'una **Activitat d'Ensenyament-Aprenentatge (AEA)** sobre els fonaments del pensament computacional, dissenyada per a l'alumnat de primer curs de CFGM SMX. L'activitat combina teoria, pràctica amb Python i mecàniques de **gamificació** per fomentar la motivació i l'aprenentatge autònom.

**Durada:** 8 hores de sessions pràctiques

---

## 🎯 Continguts de l'Activitat

L'AEA treballa el **Resultat d'Aprenentatge RA1** (CA 2.1, 2.2, 2.7 i 2.8):

- ⚙️ Ús d'IDE i entorns d'execució (Google Colab)
- 🔀 Estructures condicionals (`if`, `elif`, `else`)
- 🔁 Estructures de repetició (`for`, `while`)
- 🐛 Detecció i correcció d'errors bàsics (debugging)

## 🎮 Sistema de Gamificació

Els exercicis es classifiquen en tres nivells d'insígnia:

| Nivell | Insígnia | Descripció |
|---|---|---|
| Bàsic | 🐣 **Baby Chicken** | Consolidació de sintaxi i conceptes inicials |
| Mitjà | ⚔️ **Young Warrior** | Requereix concentració i relacionar conceptes |
| Avançat | 🐉 **Green Dragon** | Reptes complexos per potenciar el pensament lògic |

## 📊 Avaluació

L'avaluació combina diferents instruments per valorar el procés i el resultat:

- **Pràctiques de programació** classificades per nivell
- **Rúbriques** per valorar la qualitat del codi
- **Observació directa** i llistes de control
- **Proves pràctiques** amb temps limitat

### Fórmula de qualificació (RA1 · 24h)

```
QRA1 = 10·Pt1 + 10·Pt2 + 10·Pt3 + 10·Pt4 + 60·P1
```

> Es valora la **comprensió del codi**, no únicament que funcioni. L'avaluació és progressiva.
